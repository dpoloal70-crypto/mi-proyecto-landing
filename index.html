MODALIDAD DE PROGRAMACION Y DISEÑO WEB

Seccion 1006

La modalidad de programación y diseño web se enfoca en el desarrollo de páginas y aplicaciones para internet, combinando conocimientos técnicos y creativos. Por un lado, se aprende a programar utilizando lenguajes como HTML, CSS y JavaScript, que permiten estructurar, diseñar y agregar interactividad a los sitios web. Además, se introducen herramientas y tecnologías modernas como React o Vue.js, así como el uso de Git para el control de versiones. Por otro lado, el diseño web se centra en la apariencia y la experiencia del usuario. Esto incluye el diseño de interfaces (UI), la experiencia de usuario (UX), el uso adecuado de colores, tipografías y la organización de los elementos en la página. También se utilizan herramientas como Figma o Adobe XD para crear prototipos y diseños antes de programarlos. En esta modalidad también se desarrollan habilidades como la lógica de programación, la resolución de problemas y el trabajo en equipo. Asimismo, se aprende a crear sitios web responsivos, es decir, que se adapten a diferentes dispositivos como celulares, tabletas y computadoras.
